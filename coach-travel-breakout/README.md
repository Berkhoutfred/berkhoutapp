# Coach Travel — Istrië Breakout Video

Facebook breakout-ad (4:5, 1080×1350) voor **Smaak van Istrië** — 10-daagse busreis met Coach Travel.

## Output

- `output/coach-travel-istrie-breakout.mp4` — klaar voor Facebook/Instagram feed

## Opnieuw genereren

```bash
pip install -r requirements.txt
python generate_video.py
```

## Techniek

- Echte Coach Travel assets (logo, Mercedes Tourismo, Istrië-foto's van coachtravel.nl)
- Onderkant exact `#FFFFFF` (Facebook feed-wit)
- Touringcar rijdt in en overlapt nep Like/Comment/Send-knoppen
- Formaat 4:5 — optimaal voor mobiele Facebook-feed
